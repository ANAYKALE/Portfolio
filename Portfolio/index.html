MERNMarket

MERNMarket is a fully functional e-commerce application built using the MERN stack (MongoDB, Express.js, React, and Node.js). The platform allows users to browse products, add them to a shopping cart, and complete purchases with real-time updates.

Key Features:

  • Product Management: Dynamic product catalog, where users can browse products, search, and filter by categories and price. Admin functionality to add, update, or delete products using the back-end API.

  • Shopping Cart: Users can add products to a shopping cart, update quantities, and remove items. Real-time updates are displayed in the shopping cart and order summary.

  • Responsive Design: The application is mobile-responsive with a clean, user-friendly interface using React for front-end development.

  • Backend API: Built a robust RESTful API using Node.js and Express.js to handle all business logic. Implemented MongoDB as the database for storing product, user, and order information.

Blog App

This blog website is a full-featured application built using the MERN stack (MongoDB, Express.js, React, and Node.js). It provides a platform for users to create, read, update, and delete blog posts, as well as interact with the content by commenting and liking posts. The website features user authentication, making it a personalized blogging experience..

Key Features:

  • User Authentication & Authorization: Implemented secure user registration and login using JWT (JSON Web Tokens) for authentication. Different user roles: Admins can manage all posts and users, while regular users can create, edit, and delete their own posts.

  • Blog Post Management: Users can create, read, update, and delete their own blog posts. Admins can manage all posts and users through the admin panel.
